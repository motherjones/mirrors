Tests¶

class mirrors.tests.MirrorsTestCase(methodName='runTest')¶

This class adds the assertRecursiveDictContains function, which provides a simpler way to make sure one dictionary contains (at least) all of the data in another dictionary.

assertRecursiveDictContains(expected, actual)¶

Assert that one dictionary contains all of the data from another, recursive into sub-dictionaries, if necessary.

Parameters:
  • expected (dict) – The expected data
  • actual (dict) – The dictionary to check
Raises:

AssertionError
