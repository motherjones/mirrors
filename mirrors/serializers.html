class mirrors.serializers.ComponentSerializer(*args, **kwargs)¶

Used for turning a JSON blob into a mirrors.models.Component object, and back again.

restore_object(attrs, instance=None)¶

Given a dictionary of deserialized field values, either update an existing model instance, or create a new model instance.

Parameters:
  • attrs (dict) – the the key/value pairs (generally made by loading a JSON blob from the client) that represent the fields of a Component object
  • instance (mirrors.models.Component) – an optional instance of a Component. If this is set, then the values of attrs will be used update it, rather than to create a new Component.
Return type:

mirrors.models.Component
